# Sentiment Analysis on X (Twitter) Data

## 📌 Objective
This project analyzes sentiments expressed on X (Twitter) data using Natural Language Processing (NLP).

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- TextBlob
- WordCloud

---

## 📊 Features
- Sentiment Classification
- Bar Chart Visualization
- Pie Chart Visualization
- Positive WordCloud
- Negative WordCloud
- CSV Output Generation

---

## Dataset

The dataset used in this project is large in size and is not uploaded to GitHub.

You can access/download it from the link below:
https://drive.google.com/file/d/1cJlTwbx7mUxKCZS32uduCX0_850UiNKv/view?usp=sharing

## 📈 Output
The project generates:
- Positive / Negative / Neutral sentiment labels
- Graph visualizations
- Word clouds
- Processed CSV file

---

## 🚀 Conclusion
The project successfully analyzes public opinion trends from X (Twitter) data using sentiment analysis techniques.
